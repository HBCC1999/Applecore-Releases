# Applecore-Releases
A game that I made  from scratch...

# Cross-Platform
Applecore is expected to work on Windows(100%), linux and mac. Android is not supported.

# Issues
For the current update, 3.6, highscores made by user are not tracked after closing the game because it is stored in file inside the executable which is read-only.

# History
First version was released on August 2024. Though it is not available now.
Applecore 2.0 was subsequently released within a week of 1.0
Applecore 3.0 came in January 1st, 2025.
Significant updates on August 2025.
Even more significant updates in March and April of 2026.
Current update 3.6 was launched in April.
